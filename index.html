@gradientedge/akeneo-utils

Gradient Edge - Akeneo utils

An easy-to-use toolkit for working with the Akeneo API: https://api.akeneo.com/api-reference-index.html

Please note: this API is being built out bit by bit. If you're looking to use a method that doesn't appear to exist, please either raise an issue or feel free to make the change yourself and raise a PR üëç

See full documentation here: API documentation.

Installation

Simply install the @gradientedge/akeneo-utils package using your package manager, e.g.:

npm install --save @gradientedge/akeneo-utils
